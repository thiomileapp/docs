<img src="https://t3837933.p.clickup-attachments.com/t3837933/de6ae67a-75e4-4c35-bba0-bf79ca7df188/image.png" alt="" /> You could view and edit a Task by clicking this button (1), and will open a pop-up as shown below:

<img src="https://t3837933.p.clickup-attachments.com/t3837933/b81e96d3-c33b-490f-9118-f769715d6cf9/2.png" alt="" /> 1. The flow that you have selected, for the task that has been submitted, the Flow can't be changed. 2. Information that you have on this Task. You could edit this information (except for the Done task, finished task information can't be edited).

Below the Task information field there's a scrollable field that contains more task information:

<img src="https://t3837933.p.clickup-attachments.com/t3837933/3b609072-3547-4c45-9953-10fae7ab7ec9/3.png" alt="" /> This scrollable field (1) contains various information about the task, such as the task's updated time, who finished the task, where the task is finished based on latitude and longitude coordinates, photos & signature, and many more.

Please note that the information in this field is generated based on the flow setting, so the information will vary, and some are only generated after a field user finished the task.
