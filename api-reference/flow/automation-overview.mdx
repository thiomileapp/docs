## What is Automation?

Automation enables you to set up automation based on tasks' events and data source's events. It might help you automate creation of tasks, task assignment, task placement, task webhook, data source update, datasource creation, and data source webhook. Automation attributes consist of IDs, automation name, automation type, automation details, and rules.

**To see the details of automation object, please follow [this link](/api-reference/objects/automation-object)**
