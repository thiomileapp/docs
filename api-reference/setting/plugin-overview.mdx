## What is Plugin?

The Plugin API allows developers to create and manage custom add-ons within the Mile V3 platform. Plugins enable dynamic functionality through JavaScript resources and custom icons, allowing organizations to extend platform capabilities with custom features and integrations.

**To see the details of plugin object, please follow [this link](/api-reference/objects/plugin-object)**
