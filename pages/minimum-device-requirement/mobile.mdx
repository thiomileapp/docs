# Mobile

To ensure smooth operation of the MileField app, we recommend the following specifications:

## Android

- Minimum OS Version: Android 6.0 (Marshmallow):
- Minimum RAM 2GB, **recommended** 4GB

### Permissions:

1. Camera
2. Location
3. Storage
4. Telephone

## iOS

- Minimum OS Version: iOS 14:
- Minimum RAM 2GB, **recommended** 4GB

### Permissions:

1. Camera
2. Location
3. Storage
4. Telephone
