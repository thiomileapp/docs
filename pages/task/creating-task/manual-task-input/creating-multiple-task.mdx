1. Select a Flow.
2. Click **Bulk Manual**.
3. Fill in the information.  
   This field will vary based on your Flow settings.
4. You can use the default configuration from the Flow, or you can manually adjust the Start and End time.  
   The configuration can be found at [Flow Configuration, Time](https://web.mile.app/flow/flow).
5. Use the action buttons to add or delete a Task.
